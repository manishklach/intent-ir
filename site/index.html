The Deterministic Substrate for AI Agents

Moving from opaque text to verifiable intent. A complete IR layer for the next generation of agentic workflows.

View on GitHub pip install intentir

Deterministic

Every execution is recorded in a bit-perfect trace, allowing for complete replayability and auditing.

Verifiable

A static verifier ensures resource bounds and safety constraints are met before a single instruction runs.

Inspectable

Low-level IR instructions provide a clear view of agent intent, removing the "black box" problem of LLMs.

hello_agent.asm 
DECLARE_AGENT   "hello-bot"
DECLARE_TASK    "greet"
LOG             "Hello, INTENT-IR!"
COMMIT          {"greeting": "Hello World"}
HALT
